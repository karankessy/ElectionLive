# 🇳🇵 ElectionLive

A vibe-coded hobby project to visualize the Nepal Election 2082 in real time.

Built on a whim — no grand plan, just curiosity and code. This dashboard scrapes live election data and presents it through interactive visualizations so you can follow the results as they roll in.

## Features

- **Live Seat & Majority Tracker** — watch the seat count update in real time
- **Party Leaderboard** — see which parties are leading across the country
- **Province Breakdown** — drill into results by province
- **Popular Candidates & Spotlight Cards** — track high-profile races
- **Recent Changes Feed** — a live feed of the latest swings and updates
- **RSP Tracker** — dedicated tracker for RSP performance
- **Dark / Light Mode** — because vibes matter

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [SWR](https://swr.vercel.app) for data fetching
- Deployed on [Vercel](https://vercel.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it in action.
